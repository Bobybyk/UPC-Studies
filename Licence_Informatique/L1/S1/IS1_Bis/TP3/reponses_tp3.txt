Exercice 5 :
	1/. Les symboles décrivants les droits sont : xwr (x : droit d'execution ; w : droit d'écriture ; r : droit de lecture).
		Le plus gros fichier ordinaire est titi avec 32octets.

Exercice 6 :
	1/. Nous n'avons que le droit de lecture sur le fichier tata. Lorsqu'on souhaite le modifier, on ne peut pas enregistrer les modifications, nous ne sommes qu'en lecture seule.

	2/. Nous avons seulement le droit d'écriture avec titi mais pas le droit de lecture.

	3/. Nous avons les droits de lecture et d'execution du fichier toto. Nous pouvons lire son contenu, executer son contenu mais pas le modifier.

	4/. Les seuls droits que nous possédons sur le fichier tutu sont les droits d'execution. Nous pouvons donc executer son code mais pas le lire ni le modifier.
